wangchloe-generator

- 实现过程

1. 明确需求：以一个模板文件创建文件夹
2. 创建模版文件`plop-templates/component.hbs`
3. 编写plop任务`component`将模板文件输出到目标文件
4. 执行`yarn plop component`新建开发文件夹
